You are a senior UI/UX designer and frontend engineer specializing in aviation dashboards, real-time systems, and high-performance web applications.

Your task is to completely remaster the UI of my virtual airline’s flight tracking software. The goal is to transform it into a modern, sleek, highly interactive, and professional-grade interface comparable to real-world aviation tools.

### Core Objectives

* Redesign the entire UI with a clean, modern, and premium aesthetic
* Prioritize usability, clarity, and performance
* Create a visually impressive experience with smooth, purposeful animations
* Ensure the interface feels like a real-world airline operations dashboard

### Design Direction

* Style: Minimalist, glassmorphism / soft dark mode with subtle gradients
* Inspiration: Aviation ops dashboards, modern SaaS apps, flight tracking tools
* Color palette: Dark base with accent colors (e.g. cyan, blue, orange for alerts)
* Typography: Clean, highly readable, modern sans-serif
* Use depth, blur, and layering to create a premium feel

### Key Features to Include / Improve

* Real-time flight tracking map (interactive, smooth zoom/pan)
* Aircraft list panel with sorting, filtering, and search
* Flight details panel (altitude, speed, heading, route, status, ETA, etc.)
* Live updates without UI lag
* Alerts/notifications system (delays, emergencies, events)
* Timeline or playback system for flights
* Weather overlays (optional but encouraged)
* Airport and route visualization
* Mobile responsiveness and adaptive layouts

### Animations & Interactions

* Smooth transitions between panels and states
* Hover effects and microinteractions
* Animated data updates (numbers smoothly changing)
* Loading skeletons instead of spinners
* Map animations (flight path drawing, movement interpolation)
* Subtle motion that enhances UX without reducing performance

### Technical Requirements

* Use a modern frontend stack (React, Next.js, or similar)
* Component-based architecture
* Clean, maintainable, scalable code
* Optimized rendering for real-time data
* Use WebSockets or efficient polling for live updates
* Consider performance (virtualized lists, memoization, etc.)
* Accessibility (ARIA roles, contrast, keyboard navigation)
* Responsive design (desktop-first but mobile-friendly)

### Deliverables

Provide:

1. A full UI/UX redesign concept (describe layout and structure clearly)
2. A component hierarchy / architecture plan
3. Example UI code (React + Tailwind or similar preferred)
4. Styling system (colors, spacing, typography)
5. Animation approach (libraries like Framer Motion if applicable)
6. Suggestions for improving user experience beyond visuals
7. Any additional advanced features that would make this feel next-level

### Extra Expectations

* Think like a real airline operations tool, not a basic hobby project
* Go beyond what I asked and suggest innovative features
* Focus heavily on both aesthetics and performance
* Make the UI feel “alive” with real-time data

Ask any clarifying questions if needed, but otherwise proceed with a complete redesign proposal and implementation example.
